\# Project Overview



The Simple Expense Tracker is a small web-based application designed to help users track daily expenses.



The project was created as a beginner-friendly practice project for learning:



\- Git

\- GitHub

\- Spring Boot

\- Thymeleaf

\- MySQL

\- Basic CRUD operations



\## Main Purpose



The purpose of this project is to allow users to record expenses and manage them from a simple web interface.



\## Main Modules



1\. Expense Model  

&#x20;  Represents the expense table in the database.



2\. Expense Repository  

&#x20;  Handles database operations using Spring Data JPA.



3\. Expense Service  

&#x20;  Contains business logic for managing expenses.



4\. Expense Controller  

&#x20;  Connects the backend logic to the Thymeleaf pages.



5\. Thymeleaf UI  

&#x20;  Allows users to add, view, and delete expenses.
